Hexo 博客搭建教程 I

2022/4/17 教程 Hexo

我搭建这个博客搜了好多教程，都不是特别全，就写了这篇教程

0. 前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
学习 Markdown 语法可以看 Github 官方文档：基本撰写和格式语法

1. 安装 Git 和 PNPM

  • Git
    从 https://git-scm.com/downloads 下载 Git
    完成后用 git --version 命令检查，有提示即安装正确

  • PNPM（可选）

    • Windows
      在 Powershell 输入以下命令安装 PNPM
      $env:PNPM_VERSION='7.0.0-rc.2' ; iwr https://get.pnpm.io/install.ps1 -useb | iex
    • 类 POSIX 系统
      wget -qO- https://get.pnpm.io/install.sh | PNPM_VERSION=7.0.0-rc.7 sh -

    完成后用 pnpm -v 检查
    也可以不安装，但是后续的命令都要换成 NPM 版本的
    为什么要用 PNPM？ 看速度就知道了

    更详细的，可以看一下 PNPM 官方 的介绍

2. 安装 Node.js

  • 已安装 PNPM
    运行命令 pnpm env use -g lts 安装 Node.js 最新 LTS 版本（注意要在管理员模式运行）
    运行时间可能会有点长，也就二三十分钟，耐心等一下
    完成后用node -v 检查，有提示即安装正确
  • 未安装 PNPM
    从 https://nodejs.org 下载 Node.js
    完成后用 node -v 和 npm -v 两个命令检查，有提示即安装正确

3. 安装 Hexo CLI

输入命令 pnpm add -g hexo， 用 hexo -v 检查安装
新建一个文件夹，作为博客目录，cd 进入文件夹，运行 hexo init & pnpm i

Hexo 官方文档：https://hexo.io/zh-cn/docs

4. Hexo 的一些命令

  • 生成静态文件：hexo g
  • 清除静态文件：hexo cl
  • 在本地运行：hexo s
  • 部署到网站：hexo d
  • 生成静态文件并部署到网站：hexo d -g 或 hexo g -d
  • 创建新文章：hexo new <post-name> <post-name> 为文件名*

P.S.：创建新文章命令中的<post-name>是文件名，标题在文章中的 title 参数中修改，文件名建议全部小写，若有大写字母，部署后部分主题可能会 404
